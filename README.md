# AireSpring Tech Test
## Summary
This is a simple ASP.NET core project that allows to create employee records, edit and delete them in a database. 
The project includes the SQL Scripts to generate the database in a SQL Server instance. It uses T-SQL in code just to prove SQL Knowledge.

This project has been used in Tech test for AireSpring company. It's for educational purposes.

### Requirements
- .NET 7.0
- SQL Server instance (Either SQL Express, Development, or even Docker).
- The project can run in macOS and Windows (I haven't tested it on Linux).
- To connect to your database, please replace the DefaultConnection value in appsettings.json
